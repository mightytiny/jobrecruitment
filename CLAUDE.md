# Rules

- **Answer questions without taking action.** A question is not a request to write code or make changes. Only act when explicitly told to.
- **Read before touching.** Always be sure of what I am doing. Never guess or assume.
- **Do exactly what is asked. Nothing more.** No layout changes, refactors, or improvements that were not requested. Instead, ask about further changes beforehand.
- **When unsure, ask. Do not guess and act.**
