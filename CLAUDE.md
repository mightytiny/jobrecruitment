# Rules

- **Answer questions without taking action.** A question is not a request to write code or make changes. Only act when explicitly told to.
- **Read before touching.** Run `git diff HEAD` before any edit session to understand the current state.
- **Do exactly what is asked. Nothing more.** No layout changes, refactors, or improvements that were not requested.
- **When unsure, ask. Do not guess and act.**
